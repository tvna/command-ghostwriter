# Welcome to Command ghostwriter
このアプリケーションでは、設定ファイル(toml/yaml)とJinjaテンプレートファイルをアップロードして、インフラ構築コマンドなどのテキスト整形ができます。
ファイルをアップロードして、「テキスト整形の実行」をクリックして結果を確認してください。
