# Proposals

このディレクトリは協議中の事項[Why/Why Not検討中]の置き場です。
まだ決定に至っていない設計上の選択肢や変更提案を、根拠とともに記述したドキュメントを格納します。

## 運用

- 提案が決定に至ったら、内容を該当する `prd` / `standards` / `runbooks` へ昇格させ、本ディレクトリからは削除する。
- 決定の経緯 [採用/不採用の理由] は `history` へ記録する。
