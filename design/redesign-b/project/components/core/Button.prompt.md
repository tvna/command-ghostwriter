Primary action control for Command Ghostwriter — use for any clickable command, like "CLIコマンド生成" or "ダウンロード".

```jsx
<Button variant="primary" icon="📝" onClick={generate}>CLIコマンド生成</Button>
```

Variants: `primary` (brand-red CTA), `secondary` (bordered, default — turns red on hover), `ghost` (borderless). Sizes: `sm` / `md` / `lg`. Pass `fullWidth` for Streamlit's `use_container_width` look; `disabled` dims to 45%.
